> JavaScript

[cite_start]JavaScript es el lenguaje de programación que le da vida e interactividad a las páginas web[cite: 4, 5]. Mientras que HTML es el esqueleto y CSS es la piel, JavaScript es el sistema nervioso. [cite_start]Se ejecuta en el navegador del usuario y permite crear contenido dinámico, validar formularios[cite: 37], controlar multimedia y animar imágenes.

[cite_start]Con JavaScript, podemos manipular el DOM (Document Object Model) para cambiar cualquier elemento de la página sin necesidad de recargarla, creando una experiencia de usuario fluida y moderna[cite: 11]. Es una de las tecnologías fundamentales de la web.
