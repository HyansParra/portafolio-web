> Node.js

Node.js es un entorno de ejecución que nos permite correr código JavaScript fuera del navegador, es decir, en el servidor. Esto revolucionó el desarrollo web, ya que permitió a los desarrolladores usar un solo lenguaje (JavaScript) tanto para el frontend (lo que ve el usuario) como para el backend (la lógica del servidor).

Es ideal para construir APIs rápidas y escalables, aplicaciones en tiempo real (como chats), y microservicios. Se basa en una arquitectura orientada a eventos y no bloqueante, lo que lo hace muy eficiente para manejar múltiples conexiones al mismo tiempo.
