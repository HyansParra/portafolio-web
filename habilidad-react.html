> React

React es una biblioteca de JavaScript, creada por Facebook, para construir interfaces de usuario interactivas y reutilizables. Su principal característica es el uso de "componentes", que son como piezas de Lego que podemos combinar para crear aplicaciones complejas. Cada componente maneja su propio estado y lógica.

En lugar de manipular el DOM directamente, React utiliza un "DOM Virtual", una copia en memoria que le permite calcular los cambios de manera muy eficiente y actualizar solo lo que es estrictamente necesario en la pantalla. Esto hace que las aplicaciones de React sean extremadamente rápidas y fluidas.
