> HTML5

HTML, o "HyperText Markup Language", es el esqueleto de todas las páginas web. Es el lenguaje de marcado estándar que se utiliza para crear y estructurar el contenido de un sitio, como textos, imágenes, enlaces y formularios.

La versión 5 (HTML5) introdujo nuevas etiquetas semánticas como <header>, <footer>, <section> y <article>, que ayudan a los navegadores y a los motores de búsqueda a entender mejor la estructura del contenido. También trajo consigo soporte nativo para audio y video, y nuevas APIs para crear aplicaciones web más complejas e interactivas.
