> SQLite

SQLite es un motor de base de datos SQL que es increíblemente ligero y no requiere un servidor separado para funcionar. A diferencia de otros sistemas como MySQL o PostgreSQL, una base de datos SQLite es simplemente un archivo (generalmente con extensión .db o .sqlite) que puedes guardar en cualquier lugar de tu proyecto.

Es la base de datos más desplegada del mundo, ya que se usa en millones de aplicaciones móviles, navegadores web y otros programas. Es perfecta para proyectos pequeños, desarrollo, prototipos o aplicaciones que necesitan una base de datos portable y fácil de configurar.
